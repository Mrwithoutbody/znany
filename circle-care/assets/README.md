# Zdjęcia twórców

Wrzuć tu zdjęcie **kwadratowe** (np. 400×400 px, `.jpg`) o nazwie = klucz z tabeli.
Prototyp podmieni je automatycznie. Brak pliku = ładny fallback (gradient + inicjały),
więc nic się nie psuje.

> ⚠️ Użyj **oficjalnych / press / profilowych** zdjęć, do których manager ma prawa.
> Nie hotlinkujemy z Instagrama (IG blokuje + kwestia praw autorskich). Pobierz plik i wrzuć tutaj.
